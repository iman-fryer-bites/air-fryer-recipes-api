# Air Fryer Recipes API

A free open-source JSON API for air fryer recipes.

Features

- Air Fryer Recipes
- Cooking Times
- Temperatures
- Calories
- Categories
- Search
- Seasonal Recipes
- Kid Friendly Meals
- Family Dinners

Example

GET

/api/recipes

Response

[
 {
   "title":"Air Fryer Peanut Butter Cookies",
   "time":"10 min",
   "temperature":"350°F",
   "category":"Desserts"
 }
]
