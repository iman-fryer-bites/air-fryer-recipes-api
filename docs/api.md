# API Documentation

## Get All Recipes

GET

```
/recipes
```

Returns every recipe.

---

## Get Recipe

GET

```
/recipes/{id}
```

Returns a single recipe.

---

## Search

GET

```
/recipes?search=cookies
```

---

## Categories

GET

```
/categories
```
